## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- yarn
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/dreyzu/digital-business-card
   ```

2. Install packages

   ```sh
   yarn install
   ```

3. Start development server

   ```sh
   yarn start
   ```
